# Sportify AI - Advanced Features Guide

## 🎯 Quick Start

### 1. Visualization Dashboard
**Location**: Visible in Recommendations tab below the recommendation list

**Features**:
- **5 Different Chart Types**: Scores, Positions, Age, Market Value, Heatmap
- **Statistics Summary**: Total recs, average score, top score, top player
- **Interactive Selection**: Click chart buttons to switch between visualizations
- **Color Coding**: Green (high), Blue (medium), Yellow/Orange (low), Red (very low)

**How to Use**:
1. Generate recommendations
2. Scroll down to see visualization
3. Click chart type buttons to switch views
4. Hover over data points for details

---

### 2. Interactive Chatbot Widget
**Location**: Floating button in bottom-right corner

**Quick Actions**:
- 📖 **Explain**: "Why is the top player recommended?"
- ⚖️ **Compare**: "How do these players compare?"
- ⚠️ **Risks**: "What are the risks?"
- 📋 **Summary**: "Give me a summary"

**How to Use**:
1. Click the 💬 button to open chat
2. Use quick action buttons for instant queries
3. Or type custom questions
4. Follow-up questions auto-generated
5. 🔄 to clear chat, ✕ to close

**Capabilities**:
- Explains recommendation rationale
- Compares players in detail
- Identifies concerns and opportunities
- Generates executive summaries
- Contextual follow-up questions

---

### 3. Analytics Tab
**Location**: New tab in main navigation bar

**Analysis Types**:
- 📊 **Recommendations**: Analyze recommendation sets
- 👥 **Players**: Individual player deep-dive
- 🏢 **Clubs**: Club-level analysis
- 📈 **Market Trends**: Global market insights

**Views**:
- **Overview**: Key metrics cards
- **Visualization**: Interactive charts
- **Insights**: AI-generated insights
- **Comparison**: Comparative analysis

**How to Use**:
1. Click Analytics tab
2. Select analysis type from dropdown
3. Choose specific entity to analyze
4. Click 🔍 Analyze
5. View results in different tabs
6. Use chatbot for questions

**Actions**:
- 📄 Generate PDF report
- 📊 Switch between different views
- 💬 Ask chatbot questions about data

## 📊 Visualization Types Explained

### 1. Score Bar Chart
Shows recommendation scores as horizontal bars
- **Color**: Green (high quality), Blue (good), Yellow (fair), Red (lower)
- **Use**: Quickly see quality ranking of recommendations
- **Best For**: Comparing overall recommendation quality

### 2. Position Pie Chart
Distribution of recommended players by position
- **Color**: Different color per position
- **Use**: See squad balance and position diversity
- **Best For**: Understanding position coverage

### 3. Age Distribution Chart
Bar chart showing age distribution
- **Statistics**: Min, Max, Average age shown
- **Use**: Analyze age profile of recommendations
- **Best For**: Squad balance and development planning

### 4. Market Value Scatter Plot
X-axis: Player age, Y-axis: Score, Bubble size: Market value
- **Pattern**: Identifies value-for-money players
- **Use**: Market analysis and budget optimization
- **Best For**: Financial decision making

### 5. Multi-Factor Heatmap
Table showing Fit, Performance, and Availability scores
- **Color Intensity**: Green (high), Red (low)
- **Use**: Detailed metric comparison
- **Best For**: Technical analysis

## 🤖 Chatbot Intelligence Features

### Context Awareness
- Understands selected club context
- References recommendation data
- Tracks conversation history
- Adapts responses based on analysis type

### Explanation Capabilities
- **Why Recommendations?**: Details on player suitability
- **Comparative Analysis**: Head-to-head player comparisons
- **Risk Assessment**: Potential concerns and red flags
- **Market Context**: How players compare to market average

### Question Types Supported
```
// Player-focused questions
"Why is [player] recommended?"
"What are [player]'s strengths?"
"How has [player] performed recently?"

// Comparative questions
"Compare [player1] vs [player2]"
"Which recommendation is best?"
"What makes them different?"

// Risk questions
"What are the concerns?"
"Are there any red flags?"
"What could go wrong?"

// Summary questions
"Summarize the analysis"
"What should I know?"
"Give me the executive summary"
```

### Follow-up Generation
Automatically suggests relevant follow-up questions:
- Based on your question
- Related to the analysis
- Progressive complexity
- Different angles on same topic
